# Flashcards PWA

Eine Progressive Web App zum Erstellen und Lernen digitaler Karteikarten.

## Features

- Lernkarten erstellen
- Lernkarten bearbeiten
- Lernkarten löschen
- Lernsets verwalten
- Lernmodus mit zufälliger Reihenfolge
- Fortschrittsspeicherung
- Offline-Nutzung durch PWA

---

## Technologien

- React
- Vite
- React Router
- IndexedDB
- Dexie.js
- PWA / Workbox

---

## Setup

Repository klonen:

```bash
git clone REPOSITORY_URL
```

In den Projektordner wechseln:

```bash
cd flashcards-pwa
```

Dependencies installieren:

```bash
npm install
```

Entwicklungsserver starten:

```bash
npm run dev
```
